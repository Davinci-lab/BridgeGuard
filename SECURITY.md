# Security Policy

## Reporting a Vulnerability

BridgeGuard is a **defensive research prototype**. It does not handle private keys, execute transactions, or connect to live chains by default.

If you discover a security issue in the code (e.g., dependency vulnerability, data exposure), please report it by opening a GitHub issue with the label `security`. We do not offer a bug bounty at this time.

## Safe Use Notice

This software is not audited and must not be used as the sole control for real funds. See [README.md](README.md) for full disclaimer.
